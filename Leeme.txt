REQUISITOS NECESARIOS PARA DESPLIEGUE DEL SERVIDOR
Esta aplicación web se ha desarrollado utilizando la tecnología del lado de servidor node.js. Para desplegarla,
es necesario tener esta tecnología instalada. Si no se tiene, se puede descargar de su sitio web oficial:
https://nodejs.org/es/download/


DESPLIEGUE DEL SERVIDOR
Se lanzará la página ejecutando el siguiente comando en el directorio raíz del proyecto:
$node server.js
